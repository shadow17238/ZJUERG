# 电工电子学实验

> **课程基本信息**

- 学分：1.5
- 开课学期：秋冬、春夏
- 培养方案建议修读学期：大二春夏
- 说明：秋冬学期的主要面向飞设、车辆，春夏学期的主要面向机械、过控

> 《电工电子学》和《电工电子学实验》的关联非常紧密，建议在同一个长学期修读，尽量不要分开
>
> 与《大学物理实验》不同，《电工电子学实验》是选定一位老师后由这位老师主导整个学期的课，考核方式也由老师决定
